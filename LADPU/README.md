# LADPU Home Energy Projection (UNM Energy Econ Lab)

Interactive projection of Los Alamos residential electricity and natural gas use to 2070.

`index.html` is fully self-contained: Chart.js and all model data are embedded, so it needs
no build step, no server, and no internet connection. Put it at the root of a repository named
`ladpu` in the UNM-EEL organization and enable GitHub Pages; the tool is then served at
https://unm-eel.github.io/ladpu/ . The root host `unm-eel.github.io` is left free for the lab's
main page.
